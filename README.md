# AutoML System

This project implements a mini AutoML platform that
automatically trains and compares multiple machine
learning models to select the best-performing model.

Features:
- Automatic data preprocessing
- Multiple model training
- Hyperparameter tuning using GridSearchCV
- Model comparison
- Automatic best model selection


Models Used
- Logistic Regression
- Random Forest
- Decision Tree
- KNN
- SVM
- Gradient Boosting
- AdaBoost
- Naive Bayes
