# AutoML System Analysis

Dataset: Loan prediction dataset

Models tested:
- Logistic Regression
- Random Forest
- Decision Tree
- KNN
- SVM
- Gradient Boosting
- AdaBoost
- Naive Bayes

The AutoML system automatically trained all models,
performed hyperparameter tuning using GridSearchCV,
and selected the best model based on accuracy.

Best model: Random Forest
Accuracy: 0.89
